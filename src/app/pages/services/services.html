Servicios

¿Qué es un Servicio?

Un servicio en Angular es una clase que contiene lógica o datos que pueden ser compartidos entre diferentes componentes. Permite centralizar funcionalidades y mantener los componentes limpios.

Los servicios se suelen inyectar usando el sistema de Dependency Injection de Angular.

Crear un servicio

Usando Angular CLI, podemos generar un servicio automáticamente:

ng generate service nombre-servicio
ng g s nombre-servicio

Esto creará dos archivos: nombre-servicio.service.ts y nombre-servicio.service.spec.ts.

Ejemplo de servicio

import { Injectable } from '@angular/core';

@Injectable({
  providedIn: 'root'
})
export class MensajeService {
  obtenerMensaje(): string {
    return 'services works!';
  }
}

- @Injectable permite que Angular inyecte el servicio donde se necesite.
- providedIn: 'root' hace que el servicio esté disponible en toda la aplicación.

Usar el servicio en un componente

import { Component, OnInit } from '@angular/core';
import { MensajeService } from './mensaje.service';
@Component({
  selector: 'app-ejemplo',
  template: 'mensaje'
})
export class EjemploComponent implements OnInit {
  mensaje: string = '';

  constructor(private mensajeService: MensajeService) {}

  ngOnInit(): void {
    this.mensaje = this.mensajeService.obtenerMensaje();
  }
}

Esto mostrará en pantalla: <p>services works!</p>
