Formularios

Angular permite crear formularios de manera reactiva o template-driven. Aquí mostramos un ejemplo simple de formulario usando template-driven. 


import { NgModule } from '@angular/core';
import { BrowserModule } from '@angular/platform-browser';
import { FormsModule } from '@angular/forms';
import { AppComponent } from './app.component';

@NgModule({
  declarations: [AppComponent],
  imports: [BrowserModule, FormsModule],
  bootstrap: [AppComponent]
})
export class AppModule {}

Ahora en el componente HTML usamos ngModel para enlazar el valor del formulario: 


<form>
  <label for="nombre">Nombre:</label>
  <input type="text" id="nombre" name="nombre" [(ngModel)]="nombre">

  <button type="submit">Enviar</button>
</form>

<p>Formulario: nombre</p>

Y en el componente TypeScript: 


import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html'
})
export class AppComponent {
  nombre: string = 'formularios works!';
}

- `FormsModule` debe importarse en el módulo para usar `ngModel`. - `[(ngModel)]` permite enlace bidireccional entre el input y la propiedad del componente. - Al escribir en el input, el `

` se actualizará automáticamente mostrando el valor actual.
