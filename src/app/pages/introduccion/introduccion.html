Introducción a Angular

¿Qué es Angular?

Angular es un framework para construir aplicaciones web SPA usando TypeScript. Permite organizar la app en componentes y módulos, gestionar rutas, formularios y mantener la app escalable y reactiva.

Requisitos

  • Node.js (v18+)
  • npm (v8+)
  • Editor de código, por ejemplo VS Code

Instalar Angular CLI

Angular CLI es la herramienta oficial para crear y gestionar proyectos Angular:

npm install -g @angular/cli ng version

Crear un proyecto

Para crear un proyecto nuevo:

ng new mi-proyecto

Elige CSS si quieres usar styles.css como tu hoja de estilos global.

Levantar el proyecto

Para ver tu app en el navegador:

        ng serve
        # Abre http://localhost:4200
